## Troubleshooting

**Buttons not working?**
- Make sure file is named exactly `resume.pdf` (lowercase)
- Check it's in `assets/` folder, not `assets/images/`
- Refresh browser (Ctrl + F5)

**PDF not opening?**
- Make sure it's a valid PDF file
- Try opening the file directly to test it
- File size should be under 5MB
